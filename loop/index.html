A bad attempt at video game style audio looping– a looped track with an intro.

An audio track, specially prepared so a segment from time A to time B is repeated once after B– every S milliseconds (S being set above by 'set loop check interval'), the track will be checked if it has passed B: if so, the track is looped by (ideally) going back by B − A seconds.
